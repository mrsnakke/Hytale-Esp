# Mod de Traducción para Hytale

¡Bienvenido! Este es un mod creado para **Hytale**, diseñado para traducir automáticamente los *items* y sus descripciones dentro del juego.

Es una traduccion automatica por loque puede haber errores eres libre de mejorar la traduccion en 

## Instrucciones de instalación

Simplemente copia la carpeta completa del mod en la siguiente ubicación:

```%AppData%\Roaming\Hytale\UserData\Mods```

Una vez instalado, el mod detectará automáticamente los textos traducidos y los mostrará en español.

---

### Características principales:
- Traduce nombres e historias de ítems (armas, armaduras, consumibles, etc.)  
- Actualiza descripciones detalladas para mayor claridad  
- Compatible con versiones oficiales de Hytale  
- Fácil de usar, sin necesidad de configuraciones adicionales
